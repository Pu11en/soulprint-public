# SoulPrint Engine - Team Setup

## Quick Start (Railway)

1. **Deploy to Railway:**
   - Go to https://railway.app/new
   - Connect this GitHub repo
   - Set `SETUP_PASSWORD` in Railway env vars

2. **Open the Setup UI:**
   - Visit your Railway app URL
   - Log in with the setup password
   - Add the keys from `.env` via the Envars tab

3. **Connect Telegram:**
   - DM the bot
   - Approve pairing in Setup UI

## Keys Reference

All keys are in `.env` (gitignored, only in this private repo).
